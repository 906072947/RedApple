模块化程序设计

module TM {
	in add(int a, int b) {

	}
	
	out ans(int) {

	}

	out do(int, int) {

	}

	void print() {

	}

	void send() {

	}

	public delegate void Notify(int value); //委托，函数指针

	public static void main() {
		int p;
		TM t = new TM();
		t.ans(x -> p = x).add(10, 20);
	}
}

一个模块相当于一个特殊的类
类似Java的实现
in 和 out 是关键接口
in 相当于传入参数
out 相当于是传入一个回调函数
回调函数是非常关键的设计
